# dotfiles

## Description

My custom setup in Linux

## Requirements

- `dunst`
- `i3`
- `maim`
- `feh`
- `picom`
- `playerctl`
- `xclip`
- `xdotool`
- `rofi`
- `redshift`

```Shell
sudo apt install -y \
                    dunst \
                    feh \
                    i3 \
                    maim \
                    picom \
                    playerctl \
                    xclip \
                    xdotool \
                    rofi \
                    redshift
```

## Installation

Clone the repository and go inside the folder

```Shell
git clone https://github.com/SrVariable/dotfiles.git && cd dotfiles
```

Install the dotfiles

```Shell
./install.sh
```

> [!NOTE] 
> 
> The script has flags, you can check them out with
> ```Shell
> ./install.sh -h
> ```
